# React + TypeScript + Vite
Sample app to split bills with friends and track the dues.

Link to User stories: https://github.com/users/dileephq/projects/2

## Installation

- Clone this project

- Install dependencies
  ```bash
  $ npm install
  ```

- Run project
  ```bash
  $ npm run dev
  ```
  
- Run tests
  ```bash
  $ npm run test
  ```

## Tools & Technologies
- React
- TypeScript
- Vite
- React Testing Library
- Vitest
